# Open Script.pvf With PVF Utility

## Goal

Open the DNF 70 client `Script.pvf` package with PVF Utility 2020.

## Tool

```text
D:\PVFUTility2020\PVFUTility2020\pvfUtility.exe
```

## Target File

```text
C:\Users\tutut\Desktop\70Release\91\yuan\Script.pvf
```

## Steps

1. Double-click `pvfUtility.exe`.
2. In the top menu, click `文件(F)`.
3. Click `打开封包`.
4. Select `C:\Users\tutut\Desktop\70Release\91\yuan\Script.pvf`.
5. Wait for the package to load in the file resource manager.

## Observed Tool Output

```text
PVF Utility version: 2020.2.7.0
Requires .NET Framework 4.8
HTTP service starts on port 27000 and supports third-party access
```
