# 个人知识系统 Personal Knowledge System

这个目录是生活、工作、学习和长期提升的工作记忆。

现在这个系统也作为 AI 协作工作台使用：原始信息先进入 `00-inbox`，再由 AI 帮助拆解、处理、记录、沉淀和归档。

## 怎么使用

1. 临时想法先快速写到 `00-inbox`。
2. 每周复盘时，把有价值的内容整理到对应领域。
3. 把反复出现的问题沉淀成可复用笔记。
4. 过期或完成的资料归档到 `90-archive`。

## AI 协作方式

你可以直接把事情交给 AI：

- “帮我处理这件事”：AI 先判断目标、下一步、需要记录的位置。
- “把这段内容归档”：AI 提炼摘要、决策、行动项和后续提醒。
- “今天工作发生了这些”：AI 整理到日记、任务、会议、沟通或项目笔记。
- “复盘一下”：AI 从 inbox、任务和项目记录里整理完成项、问题和下一步。

默认流程：

1. 收集：原始内容先进 `00-inbox`。
2. 分流：按生活、工作、学习、项目、资源分类。
3. 处理：形成任务、会议纪要、决策记录、项目笔记或可复用流程。
4. 归档：完成、过期、暂停的内容进入 `90-archive` 或对应项目的归档目录。

## 目录说明

- `00-inbox`：原始想法、日记、随手记录。
- `10-life`：健康、家庭、财务、习惯、情绪、关系。
- `20-work`：工作记录、沟通、会议、决策、职业成长。
- `30-learning`：读书、课程、技术学习、语言学习、学习计划。
- `40-projects`：有明确目标的项目和长期事项。
- `50-resources`：清单、模板、资料、提示词、流程。
- `80-reviews`：周复盘、月复盘、季度复盘、年度复盘。
- `90-archive`：完成、过期或暂时不用的内容。
- `templates`：通用模板。

## 使用原则

系统要足够简单，简单到每天累的时候也愿意用。
